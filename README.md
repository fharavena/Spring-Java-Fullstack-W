# springboot-aprendiz06-MVC y Data JPA
## online lesson activitie: MVC and Data JPA

Creating a form.
Using

Run app:
```sh
$ mvnw spring-boot:run
```

Compile app:
```sh
$ mvnw package
```

Run compiled app:
```sh
$ java -jar target/nombre_aplicacion.jar
```

config application.properties if you want use mysql or h2
